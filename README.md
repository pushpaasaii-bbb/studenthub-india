# StudentHub India

StudentHub India is a modern student platform built using **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Supabase**.

## Features

* 🎓 Colleges
* 📚 Entrance Exams
* 💼 Government Jobs
* 🎁 Scholarships
* 🧮 Student Tools
* 📖 Study Materials
* 👤 Student Dashboard
* 🔐 Authentication (Supabase Ready)
* 📱 Responsive Design
* ⚡ SEO Optimized

## Tech Stack

* Next.js 14
* TypeScript
* Tailwind CSS
* Supabase
* Vercel

## Run Project

```bash
npm install
npm run dev
```

Visit:

```
http://localhost:3000
```

## Build

```bash
npm run build
```
